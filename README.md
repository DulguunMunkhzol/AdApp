# Title: Ad App

Description: 
  this is a Spring Boot service, with monolithic architecture, which demonstrates secure REST APIs and basic frontend integration.
this web application is used for uploading any ads and post it up on a forum, with a simple search function with categories in the top, 
the main users of this application is for those who want to employ, looking for employment or in need of assistance.

## Feature:
### Backend
- Full CRUD operations for core entities (Create, Read, Update, Delete) via REST API
- Search functionality for filtering/ listing entities with pagination support(Pagable)
- Persistence layer using JPA/Hibernate, supporting H2/PostgreSQL

### Frontend
- Next.js with typeScript frontend for demoing CRUD + search
  - Entity listing and search
  - Create/Update/Delete actions
